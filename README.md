## ✨ מה האפליקציה עושה?
האפליקציה מאפשרת לילדים להיות ה"במאים" של היצירה שלהם. דרך ממשק פשוט של "קוביות" ואימוג'ים, הילדים בוחרים:
1. **מי הגיבור?** (אריה, רובוט, חד-קרן או דינוזאור)
2. **מה הוא עושה?** (רוקד, אוכל, רוכב על סקייטבורד או ישן)
3. **איפה הוא נמצא?** (בג'ונגל ממתקים, על הירח, בים או בגן)
4. **סגנון מוזיקלי** (שמח, רגוע, רוקנרול או קרקס)

בסיום, האפליקציה מייצרת שני "לחשי קסם" (Prompts):
* 🖼️ **פרומפט לתמונה:** באנגלית, מותאם למודל Nano Banana.
* 🎵 **פרומפט לשיר:** בעברית, מותאם ליצירת מוזיקה ומילים ב-Gemini.

## 🛠 טכנולוגיות
* **HTML5/JS** - לוגיקה פשוטה ומהירה.
* **Tailwind CSS** - עיצוב צבעוני, מודרני ומותאם לילדים.
* **Animate.css** - אנימציות "קופצניות" שילדים אוהבים.
* **AI Collaboration** - נבנה בעזרת Gemini CLI.

## 📖 איך להשתמש בשיעור בגן?
1. פותחים את הקישור בטאבלט או מחשב המחובר למקרן.
2. מזמינים ילד/ה בכל שלב לבחור אימוג'י.
3. מעתיקים את הפרומפטים שנוצרו לכלי ה-AI.
4. מראים לילדים את התמונה שנוצרה ומשמיעים את השיר ברמקול!
